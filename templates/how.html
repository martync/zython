Couleur - The Morey Equation

La plupart des calculs demande la conversion des unités en gallons et SRM pour la couleur.
Taille du brassin : 20L = 5.28 gal
Pale malt : 5kg = 11.02 pounds / 5.9EBC = 3.0SRM
Carared : 5kg = 0.88 pounds / 40EBC = 20.0SRM


Source : http://brewwiki.com/index.php/Estimating_Color 

      (11.02*3.0) / 5.28
    + (0.88*20.0) / 5.28
    --------------------
                    9.54 MCU

    SRM = 1.4922 X (9.54^0.6859)
    = 7.03 SRM



Densité originale

Taille du brassin : 20L = 5.28 gal
Efficacité = 0.75
Pale malt : 5kg = 11.02 pounds / 5.9EBC = 3.0SRM
Carared : 5kg = 0.88 pounds / 40EBC = 20.0SRM 


      11.02 X (1.037-1)X1000 X 0,75
    +  0,88 X (1.034-1)X1000 X 0,75
    -------------------------------
                             328.24

    OG = ((328.24/5.28)/1000)+1 = 1.062



Densité finale

L'exemple est donnée pour une levure ayant un taux d'atténuation de 70%, avec un moût d'une densité originale de 1.062 (comme ci-dessus).
(Par défaut, le logiciel définira une valeur de 75% d'atténuation par défaut si aucune levure n'est renseignée). 


    FG = ((62-(0.70*62)) / 1000) + 1 = 1.019



Amertume

Si la méthode DryHop est sélectionnée, l'IBU renvoyé est 0.
Un petit tour par la conversion : Taille du brassin : 20L = 5.28 gal
Target 40.00g = 1.4oz
Strisslespalt 15.00g = 0.5oz

Nous avons aussi besoin de connaitre la densité originale = 1.062


Source : http://realbeer.com/hops/research.html 

    Target: 
    (1.65 X (0.000125^0.062) X (1-e^(-0.04 x 60)) / 4.15)  X  (0.11 X 1.4 X 7490 / 5.28)
                                                                              = 45.6 IBU

    Strisslespalt: 
    (1.65 X (0.000125^0.062) X (1-e^(-0.04 x 10)) / 4.15)  X  (0.04 X 0.5 X 7490 / 5.28)
                                                                              = 2.3 IBU

Particularités : la section ci dessous est un point à améliorer.

Si le houblon sélectionné est en pellet, l'indice trouvé sera multiplié par 1.105.
Par exemple, si notre Strisslespalt était en pellet, nous aurions :
2.3 X 1.105 = 2.5 IBU

Idem pour la méthode de "first wort hopping", l'indice est identique.
Par exemple, si nous avions ajouté notre Target en "first wort hopping", nous aurions : 45.6 X 1.105 = 50 IBU
